A Gentlish Introduction to The Axiom of Choice

A Gentlish Introduction to The Axiom of Choice¶

Primer¶

Recently, a colleague of mine asked me to try to explain the axiom of choice. The usual treatment in set theory is pretty opaque, and often couched too deeply in the language of set theory.

However, the basic idea behind the axiom of choice really isn't that challenging. If you've have some experience with formal proofs then you might have used it without knowing it.

Surjective Functions & Choice¶

What's a Surjective Function?¶

In mathematics, when introducing functions, mathematicians often write something like:

\(f : A \to B\)

In this context, \(A\) is called the domain and \(B\) is called the range.

If \(f\) was a function in a typed languages like C, C++, JAVA, or .NET languages, the domain \(A\) would be type of the input and the range \(B\) would be type of the output.

Here's the definition of a surjective function:

A surjective function (sometimes called an onto function) \(\ f: X \to Y\) is a function where for every value \(y\) in the range \(Y\), there is some value \(x\) in the domain \(X\) such that \(\ f(x) = y\).

Admittedly, that's the sort of language that turns a lot of people off to mathematics.

Fortunately, surjectivity can also be understood via pictures, and a somewhat contrived game.

Surjectivity Game¶

An example of a surjective function is \(f_0:\mathbb{R}\to\mathbb{R}\), defined to be:

\(f_0(x) = x^3 + 10 x^2 + 1\)

This function looks like this:

In [2]: 
_, ax = subplots()
x = np.linspace(-12, 6, 100)
f0 = lambda x: x**3 + 10 * x**2 + 1
ax.plot(x, f0(x), color="black")
Out[2]:
[<matplotlib.lines.Line2D at 0x112336290>]

We can see that this function is surjective by playing the following "game":

  • Player 1 picks a value \(a\) on the \(y\) axis, and draws a horizontal line \(h\)
  • Player 2 picks a value \(b\) on the \(x\) axis, and draws a verticle line \(v\)
  • If \(v\) and \(h\) meet on the curve drawn by function \(f\), player 2 wins; otherwise player 1 wins

A function is surjective if it is always possible for player 2 to win.

Here's a play of the game below on \(f_0\), where player 2 has won. Player 1 is colored red and Player 2 is colored blue:

ax.plot(x, f0(x), color="black")

There are some games where Player 2 can't win, however.

Consider the polynomial \(f_1:\mathbb{R}\to\mathbb{R}\):

\(f_1(x) = x^2 + 1\)

This function is not surjective. To prove it, here's a play of a game where Player 2 can never win given what Player 1 has played:

Winning Strategies¶

Above, we said that a function \(f\) is surjective if Player 2 can win no matter what Player 1 plays.

There's something better than having winning potential: a winning strategy means not only can Player 2 win, she always will.

Let's calculate a winning strategy for \(f_0\).

Solving the polynomial, we find:

There are a number of solutions to \(f_0(x) = y\). Sadly, none of these solutions is real valued for all values of \(y\). That means alone, none of these can be used to by Player 2 as a winning strategy.

So Player 2's strategy is to try each of these solutions, until one of them is real (up to a small numerical error), and use that.

In [6]: 
# Construct inverse functions
f0_invs = [lambdify(y, s, modules='sympy') for s in solutions]

def strategy(y):
    "A winning strategy for Player 2 in the surjection game for f0"
    for finv in f0_invs:
        x_ = N(finv(y))
        if np.abs(im(x_)) < 10**-14:
            return re(x_)
    raise Exception('Could not find inverse for {}'.format(y))

We can quickly check that this winning strategy does the job, by playing a few games.

Strategies, Right Inverses And The Axiom of Choice¶

The winning strategy we described for the surjection game is called a right inverse in mathematics. This because we can describe a winning strategy algebraically as follows:

Given a function \(f: A \to B\), we say that \(g : B \to A\) is right inverse if and only if for all \(b\) in \(B\):

\(f(g(b)) = b\)

The strategy function from the previous section obeys this rule; for any random number b then f0(strategy(b)) == b (up to some small rounding error).

It isn't so difficult to deduce the following result

Theorem: If \(f:A \to B\) has a right inverse, then \(f\) is surjective.

Pf. Let \(g\) be a right inverse of \(f\). Select an arbitrary \(b\) in \(B\); to show that \(f\) is surjective we must find a \(a\) in \(A\) such \(f(a) = b\). But we know that \(g(b)\) in \(A\) and \(f(g(b)) = b\) by definition. QED

But what about the converse? Well, with some difficulty, I showed how to write a right inverse for the polynomial \(f_0\). With a little imagination, hard work and patience, perhaps one could construct such a program for any mathematics function, right? Or in the very least, prove that an inverse function exists, even if they have no clue how to calculate it?

No. In general, it's not possible. No one will ever prove or disprove that right inverses exist for an arbitrary surjective function. Mathematics can be consistently formulated with or without it, it does not matter.

But if we want, we can declare it to true by assuming an axiom:

Axiom of Choice (v0.1-prerelease): If \(f\) is surjective, then \(f\) has a right inverse.

Now you might think to yourself: That's it? That's the axiom of choice? What's the big deal?

The trick is that this simple assumption is logically equivalent to other, less compelling statements. I will demonstrate this in future posts.
